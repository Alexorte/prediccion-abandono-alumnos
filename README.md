# Predicción de abandono de alumnos

Proyecto de análisis predictivo orientado a identificar estudiantes con riesgo de abandono académico mediante técnicas de machine learning.  
El objetivo es analizar variables relevantes del contexto académico, detectar patrones de comportamiento y construir modelos capaces de anticipar posibles casos de abandono.

## Objetivos
- Realizar un análisis exploratorio de los datos.
- Identificar variables relevantes relacionadas con el abandono.
- Preprocesar la información para su uso en modelos predictivos.
- Entrenar y evaluar distintos modelos de clasificación.
- Comparar resultados y extraer conclusiones útiles.

## Tecnologías utilizadas
- Python
- Jupyter Notebook
- pandas
- numpy
- scikit-learn
- matplotlib
- seaborn

## Estructura del proyecto
- `Practica 1 - Limpieza, EDA y selección.ipynb`: limpieza de datos, análisis exploratorio y selección de variables.
- `Practica 1 - Pipelines y Pruebas.ipynb`: construcción de pipelines y experimentación inicial.
- `Practica 2 EDA.ipynb`: análisis exploratorio adicional del conjunto de datos.
- `Práctica 2 - Aprendizaje y selección de modelo...`: entrenamiento, comparación y selección de modelos.
- `uclm_student_train.csv`: conjunto de datos de entrenamiento.
- `uclm_student_test.csv`: conjunto de datos de prueba.
- `utils.py`: funciones auxiliares utilizadas en el proyecto.

## Proceso realizado
1. Carga y exploración inicial de los datos.
2. Limpieza y preprocesado de variables.
3. Análisis exploratorio para identificar patrones relevantes.
4. Construcción de pipelines de trabajo.
5. Entrenamiento y comparación de modelos de clasificación.
6. Selección del modelo con mejor rendimiento.
7. Interpretación de resultados y conclusiones.

## Resultados
Se aplicaron distintas técnicas de clasificación para predecir el abandono de estudiantes a partir de variables académicas y personales.  
El proyecto permite comprender el flujo completo de un problema de clasificación supervisada, desde el preprocesado y el análisis exploratorio hasta la evaluación y selección del modelo final.

## Posibles mejoras
- Probar modelos adicionales y ajustar hiperparámetros.
- Incorporar nuevas variables explicativas.
- Analizar con mayor profundidad la importancia de cada variable.
- Evaluar el rendimiento con métricas adicionales como F1-score, recall o ROC-AUC.
